# 🚀 AI Content Generator API

A production-ready AI backend service built using Spring Boot and Google Gemini.

## ✨ Features
- Generate AI content
- Summarize text
- Improve content professionally

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Spring AI
- Google Gemini API
- Swagger UI

## 📌 API Endpoints

### Generate Content
POST /api/ai/generate

```json
{
  "prompt": "Write a LinkedIn post about AI"
}
```

### Improve Content
POST /api/ai/improve

### Summarize Content
POST /api/ai/summarize

## ▶️ Run Locally
mvn spring-boot:run
## ⚡Swagger UI
http://localhost:8080/swagger-ui.html
